Security Contact Policy

This app is intended for localhost or trusted private-network use. It does not include built-in authentication and should not be exposed directly to the public internet.

If you need to report a security concern, use the contact URI published in /.well-known/security.txt. Include the environment, reproduction steps, impact, and whether the issue is local-only or remotely reachable.

No public bug bounty, SLA, or guaranteed response window is offered by default. If you self-host this app, replace the contact target with your own security contact details.
